Команди по Git які були на уроці

Ініціалізація роботи git init
Зв'язати проект з Github git config --global user.name
Зв'язати емейл git config --global email.name
Проіндексувати зміни git add .
Закомітити зміни git commit --m "Вказуємо зміни"
Перейти до минулої версії з видаленням папок та файлів git reset --hard(і код версії)
Перейти до минулої версії поки без видалення git reset --soft(код версії)
Перевірити версію Git git --version
